# Beginner Django Projects (30)

1. Personal Blog - CRUD posts with titles, content, dates.
2. Todo List - Add, edit, complete, delete tasks with priorities.
3. Contact Form Site - Form with validation and success page.
4. Notes App - Create/search simple notes; text-only CRUD.
5. Bookmark Manager - Save URLs with titles, tags, and descriptions.
6. Recipe Box - Store recipes with ingredients and steps (no images).
7. Quote Library - Manage quotes, authors, and categories.
8. Book Tracker - Track books read/wishlist with status and rating.
9. Movie Watchlist - Add movies to watch, mark watched, rate.
10. Task Timer Log - Log tasks with duration and notes.
11. Event Listing - List upcoming events with date/time and location.
12. FAQ Manager - CRUD FAQs with question/answer pairs and ordering.
13. Simple Polls - Create polls with options, view results (no auth).
14. Link-in-Bio Page - Manage a list of profile links with ordering.
15. Job Board Lite - Post jobs with title, company, location, salary.
16. Issue Tracker - Track issues with title, description, status.
17. Idea Board - Submit ideas, upvote counter (no auth), sort by votes.
18. Glossary - Terms, definitions, category filtering.
19. Portfolio Showcase - CRUD projects with title, summary, tech stack.
20. Testimonials Wall - Collect and display testimonials (admin approves).
21. Changelog - Log releases/changes with version and date.
22. Snippet Saver - Save code snippets with language and description.
23. Daily Journal - Date-based entries, list by day, edit/delete.
24. Budget Categories - Track income/expense items by category (no auth).
25. Fitness Log - Log workouts (type, duration, notes), filter by type.
26. Habit Tracker - Habits with daily check-off and streak count.
27. Inventory Lite - Track items, quantity, and simple restock flag.
28. Simple Gallery - Image list with title/caption (use static placeholders).
29. Course Catalog - List courses with description, level, and duration.
30. FAQ + Contact - Combined FAQ page and contact form submission.
