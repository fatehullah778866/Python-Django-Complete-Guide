# Projects List (Beginner Level)

30 project ideas to cover core Django basics (models, views, templates, forms, admin, URLs, static files):

1. Personal Blog — CRUD posts with admin management.
2. Todo List — Tasks with complete/incomplete status and simple filters.
3. Contact Form Site — Landing page + validated contact form with success message.
4. Notes App — Create/edit/delete notes with tags and search.
5. Recipe Book — Store recipes with ingredients and steps; list/detail views.
6. Book Tracker — Track books to read/finished; rating and notes fields.
7. Movie Watchlist — Manage movies to watch, with genre/category filter.
8. Expense Tracker — Log expenses by category; monthly summary page.
9. Habit Tracker — Daily habits with streak counter and simple stats.
10. Event Listing — Upcoming/past events with date-based filtering.
11. FAQ Manager — Admin-manageable questions/answers, searchable front-end.
12. Quote Collection — Store and display quotes; random quote feature.
13. URL Shortener (basic) — Map short codes to original links; redirect view.
14. Polls/Voting App — Create polls with choices; vote and see results.
15. Job Board (mini) — Post jobs with company, location, and apply link.
16. Portfolio Showcase — Projects with images, tech stack, and contact link.
17. Gallery Viewer — Image uploads with captions; grid display.
18. Blog with Categories — Blog plus category filter and category pages.
19. Taggable Posts — Posts with many-to-many tags; tag filter page.
20. Simple Forum — Threads and replies (no auth); admin moderation.
21. Testimonials Page — Collect testimonials via form; approve in admin.
22. FAQ + Contact Combo — FAQs plus contact form with success page.
23. Product Catalog (no cart) — Products with price, description, and search.
24. Course List — Courses with syllabus outline and difficulty level.
25. Newsletter Signup — Capture emails; list subscribers in admin.
26. Support Ticket (basic) — Create tickets with status (open/closed).
27. Blog with Pagination — Paginated posts and simple search.
28. Markdown Notes — Notes stored as text, rendered with basic formatting (safe subset).
29. Location Directory — List of places with address and category filters.
30. Changelog/Announcements — Post announcements with publish date and pinned flag.
